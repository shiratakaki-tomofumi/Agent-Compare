# Windsurf セットアップ手順

## 使い方

1. このフォルダの内容を作業ディレクトリにコピー
2. `design/` フォルダも同じディレクトリにコピー
3. Windsurf でフォルダを開き、Cascade で実装指示を出す

```bash
cp -r results/windsurf/ /path/to/workspace/
cp -r design/ /path/to/workspace/design/
```

Windsurf が `.windsurf/rules/bizboard.md` を自動的に読み込みます。

## 構成ファイル

- `.windsurf/rules/bizboard.md` — プロジェクトルール（Markdown、6,000文字以内）
