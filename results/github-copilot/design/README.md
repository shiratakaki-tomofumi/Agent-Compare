# BizBoard — 業務管理ダッシュボードアプリ 基本設計書

## プロジェクト概要

BizBoardは、営業・案件・財務・人事の4領域を統合管理する業務管理ダッシュボードアプリケーションです。

### 目的

社内の主要業務データを一元管理し、部門横断的な意思決定を支援するダッシュボードを提供します。

### 対象ユーザー

| ロール | 説明 | 権限範囲 |
|--------|------|----------|
| Admin | システム管理者 | 全機能へのフルアクセス |
| Manager | 部門マネージャー | 担当部門のデータ管理・承認権限 |
| Member | 一般メンバー | 自身のデータ入力・閲覧 |

### 主要モジュール

1. **営業管理（Sales）** — 顧客管理、商談管理、売上ダッシュボード
2. **案件管理（Projects）** — 案件・タスク管理、進捗トラッキング
3. **財務管理（Finance）** — 経費申請・承認、収支サマリー、予算管理
4. **人事管理（HR）** — 従業員管理、部署管理、勤怠サマリー

### 想定規模

- ファイル数: 20〜30ファイル
- 中規模のフルスタックWebアプリケーション

## 設計書の読み方

1. まず本ファイル（README.md）でプロジェクト全体像を把握
2. `requirements.md` で機能要件を確認
3. `screen-specs.md` で画面構成とUI仕様を確認
4. `data-model.md` でデータ構造を確認
5. `tech-stack.md` で技術的な制約を確認
6. `quality-guidelines.md` で品質基準を確認
7. 実装時は `evaluation-prompt.md` をAIエージェントに渡す
