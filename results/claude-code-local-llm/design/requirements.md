# BizBoard — 機能要件一覧

## 1. 営業管理モジュール（Sales）

### 1.1 顧客管理
- **REQ-S01**: 顧客一覧の表示（テーブル形式、ページネーション付き）
- **REQ-S02**: 顧客の新規登録（会社名、担当者名、メールアドレス、電話番号、ステータス）
- **REQ-S03**: 顧客情報の編集
- **REQ-S04**: 顧客の削除（論理削除）
- **REQ-S05**: 顧客詳細画面（関連する商談一覧を含む）
- **REQ-S06**: 顧客名・ステータスによるフィルタリング

### 1.2 商談管理
- **REQ-S07**: 商談一覧の表示（ステータス別表示対応）
- **REQ-S08**: 商談の新規作成（顧客紐付け、商談名、金額、確度、ステータス、担当者）
- **REQ-S09**: 商談ステータスの更新（リード → 提案 → 交渉 → 成約/失注）
- **REQ-S10**: 商談の編集・削除
- **REQ-S11**: 商談詳細画面

### 1.3 売上ダッシュボード
- **REQ-S12**: 月次売上グラフの表示（棒グラフまたは折れ線グラフ）
- **REQ-S13**: 商談ステータス別の件数サマリー
- **REQ-S14**: 当月の売上目標vs実績表示

---

## 2. 案件管理モジュール（Projects）

### 2.1 案件管理
- **REQ-P01**: 案件一覧の表示（テーブル形式）
- **REQ-P02**: 案件の新規作成（案件名、説明、開始日、終了予定日、ステータス、担当部署）
- **REQ-P03**: 案件情報の編集
- **REQ-P04**: 案件の削除（論理削除）
- **REQ-P05**: 案件詳細画面（関連タスク一覧を含む）

### 2.2 タスク管理
- **REQ-P06**: タスク一覧の表示（案件単位）
- **REQ-P07**: タスクの新規作成（タスク名、説明、担当者、期限、優先度、ステータス）
- **REQ-P08**: タスクへの担当者アサイン
- **REQ-P09**: タスクステータスの更新（未着手 → 進行中 → レビュー → 完了）
- **REQ-P10**: タスクの編集・削除

### 2.3 進捗表示
- **REQ-P11**: 案件ごとの進捗率表示（完了タスク数 / 全タスク数）
- **REQ-P12**: 案件ステータス別の件数サマリー

---

## 3. 財務管理モジュール（Finance）

### 3.1 経費申請
- **REQ-F01**: 経費申請一覧の表示（自分の申請 / 全体）
- **REQ-F02**: 経費申請の新規作成（申請者、金額、カテゴリ、説明、日付）
- **REQ-F03**: 経費申請の編集（未承認のもののみ）
- **REQ-F04**: 経費申請の削除（未承認のもののみ）

### 3.2 承認ワークフロー
- **REQ-F05**: 承認待ち一覧の表示（Manager/Admin向け）
- **REQ-F06**: 経費の承認・却下操作（コメント付き）
- **REQ-F07**: 承認ステータスの表示（申請中 → 承認済/却下）

### 3.3 収支管理
- **REQ-F08**: 月次収支サマリーの表示（収入・支出・差額）
- **REQ-F09**: 予算vs実績の比較表示（部門別）
- **REQ-F10**: 経費カテゴリ別の支出内訳表示

---

## 4. 人事管理モジュール（HR）

### 4.1 従業員管理
- **REQ-H01**: 従業員一覧の表示（テーブル形式、ページネーション付き）
- **REQ-H02**: 従業員の新規登録（氏名、メール、部署、役職、入社日）
- **REQ-H03**: 従業員情報の編集
- **REQ-H04**: 従業員の削除（論理削除）
- **REQ-H05**: 従業員詳細画面

### 4.2 部署管理
- **REQ-H06**: 部署一覧の表示
- **REQ-H07**: 部署の新規作成・編集・削除
- **REQ-H08**: 部署ごとの所属人数表示

### 4.3 勤怠サマリー
- **REQ-H09**: 月次勤怠サマリーの表示（出勤日数、残業時間）
- **REQ-H10**: 従業員別の勤怠状況一覧

---

## 5. 共通機能

### 5.1 認証・認可
- **REQ-C01**: メールアドレス + パスワードによるログイン
- **REQ-C02**: ログアウト機能
- **REQ-C03**: ロールベースアクセス制御（Admin / Manager / Member）
- **REQ-C04**: 未認証ユーザーのログインページへのリダイレクト
- **REQ-C05**: ロールに応じたメニュー・操作の出し分け

### 5.2 ダッシュボード（トップ画面）
- **REQ-C06**: 各モジュールのKPIサマリーカード表示
  - 営業: 当月売上、商談件数、成約率
  - 案件: 進行中案件数、完了タスク数、遅延案件数
  - 財務: 当月支出、承認待ち件数
  - 人事: 従業員数、当月入社者数
- **REQ-C07**: 直近のアクティビティフィード（最新5件程度）

### 5.3 共通UI
- **REQ-C08**: レスポンシブデザイン（デスクトップ・タブレット・モバイル）
- **REQ-C09**: サイドバーナビゲーション
- **REQ-C10**: パンくずリスト
- **REQ-C11**: 検索機能（各一覧画面でのテキスト検索）
- **REQ-C12**: ローディング・エラー状態の表示
- **REQ-C13**: トースト通知（操作成功・エラー時）
